## Requirements

- Node.js (version 14 or higher)
- npm or yarn

## Installation

 **Install dependencies:**
```bash
npm install
```

Or using yarn:
```bash
yarn install
```

## Getting Started

To run the development server:

```bash
npm run dev
```

Or with yarn:
```bash
yarn dev
```

The site will open at `http://localhost:5173`

## Build for Production

Create a production build:

```bash
npm run build
```

Or with yarn:
```bash
yarn build
```

## Preview Production Build

Preview the built version locally:

```bash
npm run preview
```

Or with yarn:
```bash
yarn preview
```
